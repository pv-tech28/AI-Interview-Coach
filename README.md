# AI Interview Coach

An AI-powered interview preparation platform that helps users practice interviews, receive feedback, and improve their performance.

## 🚀 Features
- Mock interview simulation
- AI-based answer evaluation
- Real-time feedback system
- Score calculation based on responses
- Clean and interactive UI

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- WebSocket for real-time interaction

## 📂 Project Structure
- index.html → UI
- style.css → Styling
- script.js → Frontend logic
- server.js → Backend server

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Start server:
   node server.js

3. Open browser:
   http://localhost:3000
